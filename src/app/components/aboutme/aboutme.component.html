About Me

Mi nombre es Victor y estoy estudiando 2º año de DAW. Me gusta mucho la programación y el diseño web.

Este es mi primer proyecto Angular, he aprendido algunas cosas:

Pros:

  1. La escalabilidad de la pagina es visiblemente más rápida
  2. El código es más organizado y fácil de leer
  3. Dosificar el css hace mas facil como las tareas de posiscionamiento sin conflictop de otras reglas previamente definidas.
  4. Contenido dinámico acorde a la Api

Contras:

  1. Es un poco complicado de entender al principio
  2. Tiene muchas dependencias
  3. Es mal momento para aprender Angular porque estamos "en tierra de nadie"
  4. Las distintas formas de hacer las cosas son muy diferentes entre si y causan confusion
